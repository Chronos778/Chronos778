#### [EcoGuard-ML](https://github.com/Chronos778/EcoGuard-ML)
EcoGuard ML — AI-Powered Ecological Monitoring System

EcoGuard ML is an ecological monitoring system powered by machine learning. It replaces traditional formula-based models with data-driven algorithms to predict wildlife population changes, assess conservation risks, and provide actionable insights. Built with Python, TensorFlow, and Streamlit. Licensed under MIT.

#### [Advanced Data Visualization Playground](https://github.com/Chronos778/advanced-data-visualization-playground)
An interactive data visualization environment that merges AI and visual analytics.

Advanced Data Visualization Playground is an interactive web platform designed for data analysis and exploration. It allows users to visualize, compare, and interpret datasets through modular, dynamic charts and AI-assisted insights. Built with React, D3.js, and Python-based APIs, it provides a modern, intuitive interface for experimenting with real-world data.

#### [CurioCity](https://github.com/Chronos778/CurioCity)
A regional information app that provides location-based insights, facts, and data.

CurioCity is a location-based regional insights application that aggregates and presents real-time information about cities, towns, and regions.
It provides users with data on trends, facts, local prices, and community-sourced insights, helping people better understand their surroundings.
Built with React Native, integrated APIs, and a clean, mobile-friendly design.
